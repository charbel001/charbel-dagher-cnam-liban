1- run class : serveurWithGui
 - boutton :Run Server :Le serveur has been started et en attente de connexion
 - boutoon :_WHO : voir les users qui sont ONLINE
 - boutton:_SHUTDOWN: arreter le serveur et l'applicaiton
 - bouttono:DELETE Hist :supprimer les commentaires du serveur


2- run class : clientWithGui : pour cree un cient ou pluieurs client
 - commande :_connect <surnom> <machine> <port> pour se connecter au serveur exemple : _connect charbel localhost 6000
 - commande :_quit : se deconnnecter 
 - commande :_Who : voir les users  online
 - commande : @nom@message : envoyer un message 1-1 exemple : @rita@salut c'est charbel (rita doit etre online !!)
